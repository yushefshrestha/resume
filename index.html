Dashboard in Excel

FiberNet (FTTH) Performance Report - MoM

This FTTH Dashboard with Fictitious Month-o-Month data is built entirely on Excel.
An ODBC connection fetches data from the live server. Multiple Pivot tables and lookups have been arranged so as to reflect the respective indicators whose trends are displayed and M-o-M comparisons are shown. The colors (Red/Green) automatically changes according to the KPI comparison value (ie. Green for +ve growth, Red for -v growth.)
Dashboard displays trends for latest 6 months only. There is a sheet (with pivot tables) which contains all the KPIs for all segments and every months from inception, for the end-users to analyse and report as per their requirement.
